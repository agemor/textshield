textshield.js protects sensitive webpage text from web scrappers.

How?

Textshield handles protected text in a way that web scrappers cannot recognize. On the server-side, it encrypts original text with random seed. Then the encrypted text is decoded and rendered in raster format on client browsers. From web scrappers' perspective, Textshield-protected text will be seen as neither an HTML element with indecipherable value or distorted raster image.

Textshield is a javascript library that protects sensitive text from web bots
